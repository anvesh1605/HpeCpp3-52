# Inference Demo Tests

This folder contains a curated set of questions and expected answers for quick validation of the Aruba QA app.

Contents:

- `inference_cases.jsonl`: machine-readable test cases with answers
- `inference_cases.md`: human-readable version of the same cases
- `inference_metrics.json`: summary counts and coverage metrics

Notes:

- The set mixes product documentation and release-note questions.
- One case is a known missing-data example for the 5420 product-doc path.
- These are curated test references, not live evaluation outputs.
